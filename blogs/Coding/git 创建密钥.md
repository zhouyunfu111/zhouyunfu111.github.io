## Github(gitlab) 账号初始化

1. 由项目管理人员为你创建 GitLab 账户

2. 一封 GitLab 注册邮件会发送至你的邮箱

3. 点击设置密码,进入 GitLab 登陆页面,输入并确认你的 GitLab 账户的密码

4. 进入 GitLab 首页后,选中右上角头像下拉箭头,打开 Settings 页面

5. 选中 SSH 密钥配置,在此页面加入你的密钥 - 公钥,即 id_rsa.pub
